# generator-trigen-frontend

Yeoman generator for frontend development.

# Getting Started

Install with npm
```bash
npm i -g yo generator-trigen-frontend
```
or
```bash
yarn global add yo generator-trigen-frontend
```

Now yo can run generator
```bash
yo trigen-frontend
```
